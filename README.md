# VRender
VRender.ru — self-hosted SaaS для динамического рендеринга одностраничных приложений (React, Vue, Angular и др.). Превращает SPA в статический HTML для поисковых роботов Яндекса и Google. Включает личный кабинет, тарифы, интеграцию с Робокассой, админ-панель и API рендеринга с кэшированием в Redis.
